Lee Kuan Yew Chatbot
 Overview
This project is an AI chatbot designed to simulate the thinking and communication style of Lee Kuan Yew, known for his pragmatic, strategic, and no-nonsense leadership.

Features
- Answer questions about leadership, economy, and governance
- Uses structured prompt engineering
- Mimics a direct and strategic communication style

Tech Stack
- GPT-4o (OpenAI)
- Prompt Engineering
- (Optional) RAG Architecture (planned)

 Workflow
User Input → Prompt → GPT-4o → Response Output

 Future Improvements
- Implement Retrieval-Augmented Generation (RAG)
- Add vector database (FAISS/Pinecone)
- Improve response accuracy using real data sources

Example Prompt
"You are Lee Kuan Yew. Respond with a pragmatic, direct, and strategic tone."

Repository Structure
- /README.md → Project documentation
- /prompts → Prompt templates (optional)
- /docs → Workflow explanation (optional)

Purpose
This project demonstrates how AI can be used to replicate historical leadership perspectives and improve decision-making simulations.
